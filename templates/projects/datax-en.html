Datax

Business analytics platform developed in Django to generate dynamic reports on sales, customers and products.

About the project

Datax is an internal application created to provide companies with a visual and flexible tool for querying key metrics related to business performance. Through intuitive dashboards, advanced filters and optimized PostgreSQL queries, it allows visualizing indicators by customer, location, product or time period.

Technologies used

Django Bootstrap PostgreSQL HTMX Docker Git Gitea

Key features

  • Dynamic reports by customer and location: Interactive dashboard tables that can be filtered by date, customer or region to view revenue, quantity of products sold and more.
  • Optimized views with PostgreSQL: Use of materialized views and aggregate subqueries to improve performance in generating heavy reports.
  • Bootstrap + HTMX based interface:c HTMX was used to dynamically reload forms, filters and tables without needing to refresh the entire page.
  • Permission system for segmented access: User access control to limit available modules and visible filters according to role.c

Challenges and solutions

  • Optimization of complex queries: SQL queries were restructured using CTEs and aggregate views to ensure fast response times in tables with millions of records.
  • Scalability and form reuse: Search forms were designed to adapt to multiple reports, allowing logic and validation reuse.
  • Visual adaptability: A responsive design was achieved with clear navigation on both desktop and mobile devices.
