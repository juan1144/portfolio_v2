Web Portfolio

Personal website developed as a digital portfolio to showcase projects, skills, and technical articles.

About the Project

This project aims to showcase my technical skills through a clean, minimalist, and professional interface. The portfolio includes details of developed projects, source code links, and a technical blog. It is also multilingual and features responsive design.

Technologies Used

Django TailwindCss PostgreSQL HTMX Docker Git Github

Key Features

  • Project Portfolio: Detail page for each project with images, descriptions, technologies used, source code, and technical challenges.
  • Technical Blog: Section with articles on programming, web development, automation, and other topics.
  • Language Switching: Multilingual system with URLs separated by language (/es/, /en/).
  • Custom Design: Modern and dark-themed aesthetic inspired by minimalist style with support for light and dark mode.

Challenges and Solutions

  • Full Translation and Contextual Navigation: Implemented LocaleMiddleware, set_language, and dynamic structures to translate content, routes, and links without duplicating logic.
  • Clean and Modular Code: Modules were organized by app, following best practices in views, templates, and SCSS structure.
